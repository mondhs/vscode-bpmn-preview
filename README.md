# vscode-bpmn-preview

VS Code plugin for BPMN preview. 
Based on https://github.com/process-analytics/bpmn-visualization-js

# Install from code

* Check out project
* Open terminal
  * Run `npm i`
  * Run `npm run compile`
  * Run `npm test`
* In vscode open `extension.ts` and debug it as `VS Code Extension Development`
